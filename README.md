# sistema-control-acceso
‎🇬🇧Input validation script with logic for age, invitations, and VIP members | ‎🇪🇸Script de validación de entrada con lógica para edad, invitaciones y miembros VIP.

Descripcion:

Este script automatiza un control de acceso basado en 3 niveles de reglas:

-Restriccion de edad: Niega el acceso a menores de 18 años
-Acceso VIP: Reconoce a miembros mayores de 65 años como miembros VIP
-Validacion manual: Solocita y verifica invitaciones para adultos de 18-65 años

Se aplican los siguientes conceptos de Python:

-Entrada de datos con input().
-Declaraciones If, elif y else.
-Formateo de texto y comparación de cadenas.
